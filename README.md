# Generator File Greenskill

This is a Next.js application built with Firebase Studio, designed to streamline the creation and management of training program schemes and offers.

## Core Features

- **Secure Authentication**: Safe user login via Firebase authentication.
- **Centralized Dashboard**: A hub for managing registration schemes and offer generation.
- **Scheme Management**: Easily create and manage training schemes including name, unit code, and pricing.
- **Offer Generator**: Build training offers by linking to schemes and defining user requirements.
- **Data & Template Previews**: Visualize how scheme data appears on various backgrounds and on a standard A4 template.
- **Print & Edit**: Generate print-ready documents from templates and quickly edit existing schemes or offers.
- **Firestore Integration**: All user data and history are securely stored and managed in Firestore.

To get started, run the development server and navigate to the home page.

```bash
npm run dev
```
