# **App Name**: Generator File Greenskill

## Core Features:

- Otentikasi Login: Login pengguna yang aman melalui otentikasi Firebase.
- Dasbor: Dasbor terpusat untuk mengelola menu registrasi dan penawaran.
- Skema Registrasi: Buat dan kelola skema pelatihan termasuk nama, kode unit, dan harga.
- Pembuat Penawaran: Buat penawaran pelatihan dengan menghubungkan ke skema pelatihan dan menentukan permintaan pengguna. Menghasilkan tampilan berbasis daftar.
- Pratinjau Data: Menilai penempatan teks skema pelatihan di atas latar belakang menggunakan kode.
- Pratinjau Templat: Fungsi pratinjau untuk menampilkan data di atas templat PNG A4.
- Cetak & Edit: Hasilkan PDF dari templat, atau aktifkan data untuk pengeditan di menu registrasi atau penawaran.
- Integrasi Firestore: Simpan riwayat dan data yang dibuat untuk setiap pengguna.

## Style Guidelines:

- Warna utama: Biru muda (#B0E2FF) untuk menciptakan rasa tenang dan profesionalisme.
- Warna latar belakang: Biru sangat muda yang tidak jenuh (#E6F7FF) sebagai latar belakang yang bersih.
- Warna aksen: Hijau pucat (#C1FFC1) untuk elemen interaktif, menunjukkan pertumbuhan dan tindakan positif.
- Font tubuh dan tajuk: 'Inter', font sans-serif untuk kejelasan dan daya tarik modern.
- Desain yang bersih dan intuitif yang meminimalkan kekacauan untuk memfokuskan pengguna pada alur kerja utama.
- Gunakan ikon yang konsisten dan sederhana untuk navigasi.
- Animasi halus pada interaksi untuk mengonfirmasi tindakan dan menambah kesenangan.
